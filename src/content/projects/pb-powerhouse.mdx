## PB Powerhouse · 动力舱

基于生理动力学的全马战术推演工具

### 一、项目核心

该项目旨在将马拉松比赛中抽象的“撞墙”体验转化为可视化的数字化预演。通过 Python 仿真引擎模拟真实的代谢过程，帮助跑者在赛前制定最优的补给与配速策略。

1. 动态糖原建模：实时演算 42.195km 全程的糖原消耗曲线，精准定位 30km 后的能量缺口。
2. 肠胃吸收模拟：引入每小时 60g 的吸收上限（Absorption Cap）与 2km 的起效延迟（Delay），模拟真实的能量补给排队逻辑。
3. 多维环境修正：综合考虑环境温度、赛道爬升及 VDOT 跑力值，动态修正不同工况下的糖耗系数。

### 二、技术与设计

Vibe Coding 实践：项目采用 Vibe Coding 开发模式，利用 AI Agent 进行意图驱动的快速迭代，实现从生理公式到现代 UI 的高效转化。

### 三、核心价值

不仅是计算器，更是马拉松生存模拟器。它让跑者理解，每一根能量胶的价值不仅在于摄入，更在于吸收时机与代谢瓶颈的博弈。
